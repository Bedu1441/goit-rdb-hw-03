# Домашнє завдання №3 — DQL (SELECT-запити)

Репозиторій містить:
- SQL-файл із запитами: `queries.sql`
- Скріншоти виконання команд: папка `screenshots/`

## Завдання
1. Вибірка всіх полів з таблиці `products` та вибірка `name`, `phone` з `shippers`.
2. Обчислення середнього, мінімального і максимального значення `price`.
3. Вибір унікальних пар `category_id`, `price` з сортуванням і `LIMIT 10`.
4. Підрахунок кількості продуктів у діапазоні `price` від 20 до 100.
5. Підрахунок кількості продуктів та середньої ціни для кожного `supplier_id`.

## Виконання
Усі запити виконані у MySQL Workbench, результати — у папці.
